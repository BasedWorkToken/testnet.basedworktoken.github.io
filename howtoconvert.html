How Do I Convert 0xBitcoin to Based Work Token?

Requirements

 Mock 0xBitcoin 
First  you need Testnet Mock 0xBitcoin.  You can get testnet Mock 0xBitcoin for convertering by visiting the dAPP with a connected wallet and go to the bottom and click the button "Recieve 10 0xBTC v1 Tokens"
  
Example to Recieve Testnet 0xBitcoin

Now to convert to the new Base Work Token you simply input the number of 0xBitcoin you want to convert and click the button "Deposit"
It will have two transactions the first time an Approve and a "Deposit V1 to V2" transaction.

Example How to Deposit Testnet 0xBitcoin

Voila you now own RightsTo0xBitcoin Tokens and Base Work Tokens.  See the next section for how to convert back to 0xBitcoin Tokens

How Do I Convert 0xBitcoin to Based Work Token?

Requirements

Equal Amounts of RightsTo0xBitcoin Tokens and Base Work Tokens

First you need Base Work Tokens.  You can get testnet Base Work Tokens by mining on the contract, see Click Here to Learn how to mine or by converting 0xBitcoin in the section above into Base Work Tokens.

Second you need RightsTo0xBitcoin Tokens.  These can only be obtained by converting 0xBitcoin into Base Work Tokens and RightsTo0xBitcoin Tokens.  So do the section above first and get some RightsTo0xBitcoin Tokens.

Then once you have equal amounts of RightsTo0xBitcoin and Base Work Tokens you can convert them into 0xBitcoin Tokens
